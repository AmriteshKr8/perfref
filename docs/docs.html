HYPEREF-DOCS

LOGIN:

 Go to:

http://(serverip)/admin/login.php





  • Enter Web-Admin Username.
  • Enter Web-Admin Password.
Click Submit.

PRECOMP:

 Go to:

http://(serverip)/admin/precontest.php

Adding Users:

To add a user, scroll to User Management:




  • Enter a School Name.
  • Enter a School-Code.
  • Enter a Password.
Click on Add User.

Users Displayed in User List:





Adding Questions:

Scroll to Question Management:




  • Enter a Question Number.
  • Enter a Question.
  • Enter a Score.
Click on Add Question.

Questions Displayed in Question List:





Adding Question Info:

  • Go to the Webserver's install directory (/var/www/html/).
  • Go into pdfs folder.
  • For each question, place the pdf named (question_no.).pdf .
    ie: for Question 1 the pdf will be named "1.pdf".

Adding Testcases:

Scroll to Question Management:




  • Enter a Question Number.
  • Enter an Input.
  • Enter a Output.
Click on Add Testcase.
NOTE: Each line of a multiline Input or output must be separated with a caret(^) as shown above.

Questions Displayed in Question List:





Removing Entries:

Use the Delete button to remove entries.




DURING-COMP:

 Go to:

http://(serverip)/admin/during_contest.php

Or click "Contest Status" on the Nav-Bar.




Starting Competition:

To start press the "Start" button under Contest Status.
Check if status has changed.




Check for Start-Marker.




Ending Competition:

To end press the "end" button under Contest Status.
Check if status has changed.




Check for End-Marker.




Getting Ready for another Session:

After the contest is over, Truncate or save the submissions.
Go to Copy Table:



Saving Submissions:

To save the table, enter a Save Slot between 1-10 and click save.

Truncating Submissions:

To Truncate ie:"clear without saving the tables" click the "Truncate" button.

Scoreboard Watching:

Go to:

http://(serverip)/admin/scoreboard.php





  • Scoreboard shows SchoolCode, Total Score, the questions attempted and the reward given for each question to a team.
  • Scoreboard by question shows the top three submissions per question with their time of submission.

POST-COMP:

 Go to:

http://(serverip)/admin/batches.php

Or click "Batch Leaderboards" on the Nav-Bar.




Viewing Saves:

Under Batch Submissions, the slots are displayed as Table S(slotno.) with system start/end markers.

Removing Saves:

Go to the server or ssh into it:
  • Run Mysql:

    sudo mysql -A hyperef

  • Drop the Save:

    drop table S1;

    NOTE: Replace 1 with the SaveSlot.
